# Migracao do Site

Centralizador dos arquivos de migracao e operacao.

## Conteudo
- `MIGRACAO_SOWADS_RESULTSQUAD_MASTER.md`: guia especifico deste projeto.
- `MIGRACAO_SQUARESPACE_PARA_WORDPRESS_MASTER.md`: playbook geral.
- `deploy_ssh.py` / `deploy_ssh_mv.py`: deploy via SSH.
- `ACESSOS_E_OPERACAO.md`: como acessar e operar.
- `PLANO_AGENTE_ARTIGOS_LOTE.md`: plano para criar/importar artigos em massa.
- `.env.example`: modelo de variaveis.

## Seguranca
- Nao comite credenciais reais.
- Use `.env.local` no ambiente local.
